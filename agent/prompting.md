# Prompting Guide

你可以用下面的方式向 Codex 提问。

## Capture

- “请把下面这段信息先整理到 inbox，并判断后续应该归档到哪里”
- “请把这次对话中的稳定结论更新到 memory”

## Review

- “请基于当前仓库内容做一次 review，聚焦进展、阻塞和下一步”
- “请帮我做今天的 daily review”
- “请帮我做一次 session review，并告诉我哪些内容应该回写到 memory”

## OKR

- “请根据最近的 review 和 memory 起草本季度 OKR”
- “请帮我更新本季度 OKR check-in”
- “请根据季度执行情况写一版 OKR review”

## Knowledge

- “请把这条方法论整理到 knowledge”
- “请帮我判断这条内容更适合进 knowledge 还是 memory”
