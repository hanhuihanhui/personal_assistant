# Skills

这个文件记录仓库中需要长期告诉 Agent 的 skill 信息。

## Current Skills Catalog

第一阶段先使用“文档 skill”而不是运行时代码 skill：

- `Capture`: 把碎片输入转成可归档记录
- `Summarize`: 把多条记录整理成稳定记忆
- `Review`: 对阶段性进展做结构化回顾
- `Plan OKR`: 从 memory 和 review 起草季度目标
- `Link Context`: 为相关文档补充 `related` 链接

## Skill Notes

- 每个 skill 的输入输出都应通过模板约束
- skill 触发后优先复用已有文档而不是新建重复上下文
