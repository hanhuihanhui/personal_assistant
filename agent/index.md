# Agent Index

这个目录面向 Codex/Agent，定义“在这个仓库里如何工作”。

## Core Files

- `session-start.md`: 每次进入仓库的首选读取文件
- `workflows.md`: 标准工作流
- `tools.md`: 当前工具说明与边界
- `skills.md`: skill 信息与适用场景
- `prompting.md`: 推荐提问方式

## Suggested Read Path

1. 读取 `session-start.md`
2. 结合用户问题选择 `memory/`、`reviews/`、`okr/` 或 `knowledge/`
3. 按 `workflows.md` 中的流程执行
