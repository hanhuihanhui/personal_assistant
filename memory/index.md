# Memory Index

这里存放与你本人、正在推进的事项和长期上下文强相关的记忆。

## Domains

- `areas/projects/`: 项目背景、决策、风险、下一步
- `areas/goals/`: 长期目标和方向
- `areas/workstyle/`: 工作偏好和判断标准
- `areas/relationships/`: 协作关系上下文
- `areas/knowledge/`: 更像个人工作记忆的知识

## Current Canonical Files

- `areas/projects/project-personal-assistant.md`
- `areas/goals/personal-operating-system.md`

## Update Rule

如果一条信息在未来多次决策中都可能被引用，它更适合进入 `memory/`。
